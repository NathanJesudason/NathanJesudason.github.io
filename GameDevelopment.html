    The first solo game I made for a game jam, I had about a week to develop Puzzl3.

    The most impressive code to me was the shadow clone system in the game. The main way it works is by deep copying a list of position vectors of the player, and then looping through them. The main way to improvement upon this system would've been to make sure that the vectors were recorded and played at the same rate, which could've been improved by using FixedUpdate() instead of Update(). In terms of game design, I definitely think the player needed some basic animation, and well as having background music in the game.

    After a two year haitus on game development, I was the sole programmer on a team of 4 for a two week long game jam. The result was Tentacles.

    From developing this game I got reaquiantted with many features of Unity. I learned about animations and the infinite state machine that Unity uses for them. I learned about the 2D physics engine of Unity, and I first started using the particle system in Unity.
